== Description ==

Plugin loan calculators  ‘Wpcalc Loan’ for CMS WordPress – is an easy way to integrate to the website online payment of the monthly payment on the set parameters.
The plugin contains 3 Credit calculator:

* Consumer loan calculator;
* Mortgage calculator;
* Calculator of car loans.

Calculator loan monthly payment on the basis of all the data on the loan form the repayment schedule with the ability to print.

Calculators can be easily integrated into a page or post website running on CMS WordPress. On the settings page the plugin ‘Wpcalc Loan’, may set the initial data for the calculation of credit

== Installation ==

1. Upload `wpcalc-loan` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the post, you will see the icon 'Wpcalc' to select the calculator

Setting a standard plug. Calculators are placed using shortcodes:

Credit – [wpcalc-loan id = 1]
Mortgage – [wpcalc-loan id = 2]
Auto – [wpcalc-loan id = 3]
